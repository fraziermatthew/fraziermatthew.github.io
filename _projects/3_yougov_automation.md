Designed an event-driven workflow that automated YouGov analysis pipelines previously run manually in notebooks.

The system used AWS services to coordinate asynchronous processing and exposed results through frontend and visualization layers for easier consumption by internal users.

**Tech:** AWS Lambda, S3, SNS, SQS, React, Next.js, Streamlit
